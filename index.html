Так открывать нельзя (двойной щелчок по файлу).
Нужен сайт в интернете: один раз выложите папку на GitHub Pages или Netlify.
Инструкция: файл docs/КАК_ВЫЛОЖИТЬ_САЙТ.md или ОТКРЫТЬ_САЙТ.txt на рабочем столе в папке 095.
Для проверки на ПК: http://127.0.0.1:5500 (сначала запустите python -m http.server 5500)
